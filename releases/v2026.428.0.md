# v2026.428.0

> Released: 2026-04-28

## Highlights

- **Vietnamese localization (Tiếng Việt)** — Full i18n infrastructure with i18next, `i18next-browser-languagedetector`, and HTTP-backend lazy loading. Language is detected from the browser and persisted in `localStorage`. Sixteen translation namespaces cover every page and component in the UI. ([PR #2](https://github.com/saucevn/paperclip/pull/2))
- **CI branch alignment** — All four GitHub Actions workflows (`pr.yml`, `docker.yml`, `refresh-lockfile.yml`, `release.yml`) updated to target the `main` branch instead of `master`, unblocking automated canary and Docker publishing.

## Improvements

- **Locale-aware date and number formatters** — New `formatters.ts` helpers select `vi-VN` or `en-US` locale depending on the active language, with USD amounts kept in `en-US` to avoid confusing formatting for Vietnamese users.
- **Complete EN/VI key parity** — All 16 namespace files have identical key structures across English and Vietnamese; missing keys added to `costs.json`, `issues.json`, `settings.json`, and `common.json` so no fallback gaps remain.
- **Hardcoded string cleanup** — Replaced remaining hardcoded English strings in `IssueBlockedNotice`, `PropertiesPanel`, `BudgetSidebarMarker`, `ModeBadge`, and `CompanySettingsNav` with `t()` calls.

## Fixes

- **Release pipeline unblocked** — `release.yml` push trigger now fires on `main` (was `master`), enabling canary auto-publish on every merge to `main`.
- **Docker push unblocked** — `docker.yml` push trigger updated to `main` so container images are built and pushed to GHCR on every merge.
- **Lockfile refresh unblocked** — `refresh-lockfile.yml` scheduled job now targets `main`.

## Upgrade Guide

No database migrations in this release. No breaking API or config changes. The i18n files are served statically from `/locales/` and require no server-side changes. Language preference is stored in `localStorage` under the key `paperclip.language` and defaults to the browser's preferred language with a fallback to English.
